# BackEnd

## MVP
### 1. live play
현재 진행 중인 경기 정보를 알 수 있다.
### 2. 팀/선수 등록
자신만의 팀을 생성하거나 선수로 등록하여 용병 또는 팀에 가입하여 경기 참여 자격을 얻는다.
### 3. 이적 시장
#### 선수 - 등록 순, 포지션 별, 지역 별, 티어 별
선수는 자신을 이적 시장에 등록하여 팀에게 가입 제안을 받거나 용병 제안을 받는다. 
#### 팀 - 모집 인원
팀은 이적 시장을 통해 팀을 모집하거나 팀 가입 요청 및 용병 제안을 한다.
